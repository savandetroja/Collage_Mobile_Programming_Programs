# Menus Option Context

This project demonstrates the implementation of **Menus Option Context** in Android. 

## Included Files

- `MainActivity.java`
- `MainActivity2.java`
- `activity_main.xml`
- `activity_main2.xml`
- `contextmenu_list.xml`

## Setup

1. Copy the Java classes into your project's equivalent package folder.
2. Copy the XML files into your `res/layout` directory.
3. Ensure the Activity names in your `AndroidManifest.xml` match the Java files included.
