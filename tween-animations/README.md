# Tween Animations

This project demonstrates the implementation of **Tween Animations** in Android. 

## Included Files

- `TweenAnimationActivity.java`
- `activity_tween_animation.xml`
- `alpha.xml`
- `rotate.xml`
- `scale.xml`
- `translate.xml`

## Setup

1. Copy the Java classes into your project's equivalent package folder.
2. Copy the XML files into your `res/layout` directory.
3. Ensure the Activity names in your `AndroidManifest.xml` match the Java files included.
